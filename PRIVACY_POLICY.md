TabLedger is a Chrome extension that helps users save and restore tab snapshots.

## Data we collect
TabLedger does **not** collect personal information such as your name, email, passwords, messages, or location.

TabLedger only accesses the following information from your browser tabs:
- Tab title
- Tab URL
- Favicon URL (when available)

This data is used only to create and restore saved tab snapshots.

## Where data is stored
All snapshot data is stored **locally on your device** using `chrome.storage.local`.

TabLedger does **not** send any data to external servers.

## Data sharing
TabLedger does **not** sell, rent, or share user data with any third party.

## Data deletion
Users can delete saved snapshots at any time using the extension dashboard.
Removing the extension will also remove all locally stored snapshot data.

## Contact
If you have any questions about this privacy policy, you can open an issue on the TabLedger GitHub repository.
